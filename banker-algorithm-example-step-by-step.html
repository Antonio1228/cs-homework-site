Banker's Algorithm Example Step by Step (Safety Algorithm)

Banker's Algorithm is a deadlock avoidance algorithm. It checks whether the system is in a safe state by finding a safe sequence of processes.

Given

We have 5 processes (P0 to P4) and 3 resource types (A, B, C).

Allocation Matrix

Process A B C
P0 0 1 0
P1 2 0 0
P2 3 0 2
P3 2 1 1
P4 0 0 2

Max Matrix

Process A B C
P0 7 5 3
P1 3 2 2
P2 9 0 2
P3 2 2 2
P4 4 3 3

Available Vector

                Available = (3, 3, 2)

Step 1: Calculate Need Matrix

                Need = Max - Allocation
Process A B C
P0 7 4 3
P1 1 2 2
P2 6 0 0
P3 0 1 1
P4 4 3 1

Step 2: Safety Algorithm (Find a Safe Sequence)

We try to find a process whose Need is less than or equal to Available. If found, we assume it finishes and releases its Allocation back to Available.

Initial

                Work = Available = (3, 3, 2)
                Finish = [false, false, false, false, false]

Check Processes

Pick P1

                Need(P1) = (1, 2, 2) <= Work(3, 3, 2)  -> Yes
                Work = Work + Allocation(P1) = (3, 3, 2) + (2, 0, 0) = (5, 3, 2)
                Finish[P1] = true
                Safe Sequence: P1

Pick P3

                Need(P3) = (0, 1, 1) <= Work(5, 3, 2)  -> Yes
                Work = Work + Allocation(P3) = (5, 3, 2) + (2, 1, 1) = (7, 4, 3)
                Finish[P3] = true
                Safe Sequence: P1 -> P3

Pick P4

                Need(P4) = (4, 3, 1) <= Work(7, 4, 3)  -> Yes
                Work = Work + Allocation(P4) = (7, 4, 3) + (0, 0, 2) = (7, 4, 5)
                Finish[P4] = true
                Safe Sequence: P1 -> P3 -> P4

Pick P0

                Need(P0) = (7, 4, 3) <= Work(7, 4, 5)  -> Yes
                Work = Work + Allocation(P0) = (7, 4, 5) + (0, 1, 0) = (7, 5, 5)
                Finish[P0] = true
                Safe Sequence: P1 -> P3 -> P4 -> P0

Pick P2

                Need(P2) = (6, 0, 0) <= Work(7, 5, 5)  -> Yes
                Work = Work + Allocation(P2) = (7, 5, 5) + (3, 0, 2) = (10, 5, 7)
                Finish[P2] = true
                Safe Sequence: P1 -> P3 -> P4 -> P0 -> P2

Result

                System is in a SAFE state.
                One safe sequence is:
                P1 -> P3 -> P4 -> P0 -> P2

Conclusion

Banker's Algorithm avoids deadlocks by granting resource requests only if the system remains in a safe state. The key steps are computing the Need matrix and checking for a safe sequence using the safety algorithm.
