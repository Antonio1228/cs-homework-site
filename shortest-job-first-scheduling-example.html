Shortest Job First Scheduling Example (Step by Step)

Shortest Job First (SJF) is a CPU scheduling algorithm that selects the process with the smallest burst time first. This example uses the non-preemptive SJF algorithm.

Given Processes

Assume all processes arrive at time 0.

Process Burst Time
P1 6
P2 8
P3 7
P4 3

SJF Order

        Smallest burst time first:
        P4 (3) -> P1 (6) -> P3 (7) -> P2 (8)

Gantt Chart

        0    3    9    16   24
        | P4 | P1 | P3 | P2 |

Completion Time

Process Completion Time
P4 3
P1 9
P3 16
P2 24

Turnaround Time

        Turnaround Time = Completion Time - Arrival Time
        Arrival Time = 0 for all processes
Process Turnaround Time
P4 3
P1 9
P3 16
P2 24

Waiting Time

        Waiting Time = Turnaround Time - Burst Time
Process Waiting Time
P4 3 - 3 = 0
P1 9 - 6 = 3
P3 16 - 7 = 9
P2 24 - 8 = 16

Average Waiting Time

        (0 + 3 + 9 + 16) / 4 = 28 / 4 = 7
        Average Waiting Time = 7

Average Turnaround Time

        (3 + 9 + 16 + 24) / 4 = 52 / 4 = 13
        Average Turnaround Time = 13

Conclusion

Non-preemptive SJF minimizes average waiting time compared to many other scheduling algorithms, but it may cause starvation for long processes if short processes keep arriving.
