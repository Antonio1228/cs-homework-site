FIFO Page Replacement Example (Step by Step)

FIFO (First In First Out) page replacement algorithm replaces the oldest page in memory when a new page needs to be loaded.

Given

Number of Frames = 3
Reference String = 1 2 3 4 1 2 5 1 2 3 4 5

Conclusion

FIFO replaces the oldest loaded page first. Although simple to implement, it may suffer from Belady's anomaly, where increasing the number of frames increases the number of page faults.
