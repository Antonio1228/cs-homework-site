Shift Reduce Parsing Example Step by Step

Shift reduce parsing is a bottom-up parsing technique used in compiler design to analyze the syntax of programming languages.

Grammar

                E → E + T
                E → T
                T → T * F
                T → F
                F → ( E )
                F → id

Input String

                id + id * id

Parsing Steps

Stack Input Action
$ id + id * id $ Shift
$ id + id * id $ Reduce F → id
$ F + id * id $ Reduce T → F
$ T + id * id $ Reduce E → T
$ E + id * id $ Shift
$ E + id * id $ Shift
$ E + id * id $ Reduce F → id
$ E + F * id $ Reduce T → F
$ E + T * id $ Shift
$ E + T * id $ Shift
$ E + T * id $ Reduce F → id
$ E + T * F $ Reduce T → T * F
$ E + T $ Reduce E → E + T
$ E $ Accept
