Round Robin Scheduling Example with Calculation

Round Robin Scheduling is a CPU scheduling algorithm in which each process is assigned a fixed time quantum in a cyclic order.

Given Processes

Process Burst Time
P1 24
P2 3
P3 3

Time Quantum

                            Time Quantum = 4

Scheduling Sequence

                            P1 → P2 → P3 → P1 → P1 → P1 → P1 → P1

Completion Time

Process Completion Time

P2 7

Turnaround Time

                            Turnaround Time = Completion Time - Arrival Time
Process Turnaround Time

Waiting Time
